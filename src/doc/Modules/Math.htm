Math

This module contains a set of standard math functions.

Functions

float MathFloor(float val)
Description
Rounds val downward. For example, '1.75' will round to '1.0', and '-1.75' to '-2.0'.

float MathCeil(float val)
Description
Rounds val upwards. For example, '1.75' will round to '2.0', and '-1.75' to '1.0'.

float MathAbs(float val)
Description
Returns the absolute value of a number, this is, its positive value.

float MathSqr(float val)
Description
Returns the square root of a value.

float MathSin(float val)
Description
Returns the sine of an angle, in degrees.

float MathCos(float val)
Description
Returns the cosine of an angle, in degrees.

float MathTan(float val)
Description
Returns the tangent of an angle, in degrees.

float MathASin(float val)
Description
Returns the arc sine of an angle, in degrees.

float MathACos(float val)
Description
Returns the arc cosine of an angle, in degrees.

float MathATan(float val)
Description
Returns the arc tangent of an angle, in degrees.

float MathATan2(float x, float y)
Description
Returns the angle between the positive x-axis and a vector from the point (0,0) to the point (x,y), in degrees.

float MathExp(float val)
Description
Returns the exponential value of the parameter.

float MathLog(float val)
Description
Returns the natural logarithm of the parameter.

float MathLog10(float val)
Description
Returns the logarithm base 10 of the parameter.

int MathRand(int min, int max)
Description
Returns a pseudo-random number in the range from min to max.

void MathRandSeed(int seed)
Description
Initialize random number generator.
