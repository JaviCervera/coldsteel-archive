Drawing

These functions are used to draw 2D stuff to the display.

Functions

int GetColor(int red, int green, int blue, int alpha)
Description
Generates a color value from a set of red, green, blue, alpha components. Each component ranges from 0 to 255.

int GetRed(int color)
Description
Returns the red component of the specified color, in the range 0 - 255.

int GetGreen(int color)
Description
Returns the green component of the specified color, in the range 0 - 255.

int GetBlue(int color)
Description
Returns the blue component of the specified color, in the range 0 - 255.

int GetAlpha(int color)
Description
Returns the alpha component of the specified color, in the range 0 - 255.

void SetColor(int color)
Description
Sets the color in which all 2D operations will be drawn.

void Viewport(int x, int y, int w, int h)
Description
Defines a rectangular region of the screen to which all the 2D operations will be limited.

void DrawLine(int x, int y, int x1, int y1)
Description
Draws a line using the specified screen coordinates.

void DrawRect(int x, int y, int w, int h)
Description
Draws a rectangle in the specified area of the screen.

void DrawTexture(int tex, int x, int y)
Description
Draws a texture to the specified screen coordinates.

void DrawText(int font, string text, int x, int y)
Description
Draws a text in the screen.

Parameters
font - Handle to the font you want to use to draw the text, or '0' to use the default font.
text - A string containing the text you want to draw.
x, y - Screen coordinates to which the text will be drawn.

int TextWidth(int font, string text)
Description
Returns the width of the text using the specified font.

int TextHeight(int font, string text)
Description
Returns the height of the text using the specified font.
