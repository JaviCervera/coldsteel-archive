String

Although many programming languages already come with a nice set of functions to manage strings, ColdSteel includes a generic set to use with any programming language.

Functions

int StringToInt(string str)
Description
Returns the specified string as an integer value. The string must contain a number to be converted. Strings which do not start by a number, will be returned as '0'.

float StringToFloat(string str)
Description
Returns the specified string as an floating point number. The string must contain a number to be converted. Strings which do not start by a number, will be returned as '0.0'.

string StringFromInt(int number)
Description
Returns a string containing the specified integer value.

string StringFromFloat(float number)
Description
Returns a string containing the specified float point value.

string StringLeft(string str, int number)
Description
Returns the number leftmost characters of the specified string.

string StringRight(string str, int num)
Description
Returns the number rightmost characters from the specified string.

string StringMid(string str, int pos, int num)
Description
Returns num characters from the string, starting form index pos (must be bigger than 0).

string StringReplace(string str, string find, string replace)
Description
Replaces any occurances of find found in the string with replace.

int StringFind(string str, string find, int offset)
Description
Returns the index of the first occurance of find inside the string, starting the search at position offset.

string StringUpper(string str)
Description
Returns the specified string converted to uppercase.

string StringLower(string str)
Description
Returns the specified string converter to lowercase.

string StringTrim(string str)
Description
Returns a string without leading nor trailing spaces.

string StringChar(int ascii)
Description
Returns the character representation of the specified string.

int StringAscii(string str)
Description
Returns the ascii value of the specified character.

int StringLen(string str)
Description
Returns the length in characters of the specified string.

string StringField(string str, string delimiter, int index)
Description
Returns the index element in the string using delimiter as separator between elements. For example, StringField("Hello from ColdSteel", " ", 3) would return "ColdSteel".
