Core

ColdSteel SDK contains some functions to initialize and deinitialize the engine properly. Before calling any other function in the engine, you must call LibInit(). For a clean shutdown, call LibFinish() before exiting.

Functions

void CoreInit()
Description
Initialises the engine. You must call this function at the begining of your program, before calling any other ColdSteel functions.

void CoreFinish()
Description
Cleanly shuts down the engine, releasing all the memory allocated by it. After calling this, you can exit your program safely.

void CoreMessage(string msg, int error)
Description
Shows as popup message. The 'error' parameter tells if the message should contain an error icon.

int CoreTimer()
Description
Returns the amount of milliseconds elapsed since the system started.
