Joystick

Although most PC games are controlled through the keyboard and mouse, being able to use a joystick is a really good option, as many games are better controlled this way. ColdSteel comes with a set of functions to manage analog joysticks (the same functions can be used to handle classic digital pads). Up to two joysticks are supported, numbered 0 and 1.

Functions

string JoyName(int id)
Description
Returns the name of the specified joystick. If the joystick is not available, an empty string will be returned.

int JoyButton(int id, int button)
Description
Returns '1' if the specified joystick button is pressed, or '0' otherwise. Joystick button indices start at '1'.

float JoyAxis(int id, int axis)
Description
Returns the position of the specified axis (a value from -1.0 to 1.0). Joystick button indices range form '1' to JoyNumAxes().

int JoyNumAxes(int id)
Description
Returns the number of axes available for the specified joystick.
