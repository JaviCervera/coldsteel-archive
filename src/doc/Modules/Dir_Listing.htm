Dir Listing

A set of functions to search for files within directories is provided.

Functions

void PackageAdd(string pak)
Description
Adds a package to the resource search path. When you load a file or any media resource, the engine will look into the path specified, and if the file is not found, it will look into the packages added. This is a nice way to reduce disk usage. A package is a standard zip file.

int DirList(string dir)
Description
Opens a directory for listing its files, and returns a handle to it.

void DirClose(int dir)
Description
Closes a directory after we have finished listing its files.

int DirFileCount(int dir)
Description
Returns the number of files and directories found in the specified dir.

string DirFileName(int dir, int index)
Description
Returns the name of the file at the specified index in the specified dir

int DirFileIsDir(int dir, int index)
Description
Returns '1' if the specified file is a directory, or '0' otherwise.
