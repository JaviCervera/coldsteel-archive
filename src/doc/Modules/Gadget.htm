Gadget

A gadget is any type of element (a window, a button, a menu...) inside the user interface of your game. You can create any type of gadget using the functions in the 'Gui' module, and modify your gadgets later with these functions.

Functions

void GadgetFree(int gadget)
Description
Frees a previously created gadget.

int GadgetType(int gadget)
Description
Returns the type of the specified gadget. It will return a value from the list of gadget types listed above.

void GadgetEnable(int gadget, int state)
Description
Enables or disables a gadget.

Parameters
state - '1' to enable a gadget (the intial default state), '0' to disable it.

string GadgetGetText(int gadget)
Description
Returns the text of a gadget (the text of a button, or the title of a window, for example).

void GadgetSetText(int gadget, string text)
Description
Set the text of a gadget (the text of a button, or the title of a window, for example).

void GadgetShape(int gadget, int x, int y, int w, int h)
Description
Change the position and size of a gadget.

void GadgetVisible(int gadget, int state)
Description
Set the visible mode of a gadget.

Parameters
state - '1' to make the gadget visible (the initial default state), '0' to make it invisible.

int GadgetChildren(int gadget)
Description
Returns the number of children that the gadget has (for example, the number of gadgets contained in a window).

int GadgetChild(int gadget, int index)
Description
Returns the handle of the children with the specified index.

int GadgetParent(int parent)
Description
Returns the parent handle of the specified gadget.

void GadgetSetImage(int gadget, int tex, int x, int y, int width, int height, int alpha)
Description
Set a texture image for the gadget. you can specify if you want to use the alpha channel in the texture or not.

void GadgetSetFont(int gadget, int font)
Description
You can pass a font to use by the gadget, or set to '0' to use the skin default font.

void GadgetSetColor(int gadget, int color, int override)
Description
You can set override to '1' and specify one color to use instead of the skin default color. Set override to '0' to use the skin color.

void GadgetSetState(int gadget, int state)
Description
Set the state of the gadget. In most gadgets, this value can either be '1' or '0', But on scrollbar gadgets this is used to set the current value of the scrollbar.

int GadgetGetState(int gadget)
Description
Returns the state of the gadget.

int GadgetAddItem(int gadget, string text, int id)
Description
Adds a new item to the gadget.

int GadgetGetItemId(int gadget, int index)

string GadgetGetItemText(int gadget, int index)

int GadgetSelectedItem(int gadget)

void GadgetSelectItem(int gadget, int index)

int GadgetCountItems(int gadget)

void GadgetSetItemText(int gadget, int index, string text)

void GadgetEnableItem(int gadget, int index, int enable)

void GadgetRemoveItem(int gadget, int index)

void GadgetRemoveItems(int gadget)
