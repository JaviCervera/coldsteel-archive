Effect

Apart from simple shaders, ColdSteel supports effects. An effect is a complex shader, which can be composed of several techniques and passes, and is stored in a standard DirectX .FX file.

Functions

int EffectRegister(string effect, int base_mat)
Description
Creates a new effect.

Returns
The handle of the registered effect. This handle can then be used as a standard material type.

Parameters
effect - A string containing the source code of the effect.
base_mat - The base material type of this effect.

int EffectRegisterFile(string effect_file, int base_mat)
Description
Loads a new effect from a file.

Returns
The handle of the registered effect. This handle can then be used as a standard material type.

Parameters
effect_file - The file containing the effect. This is a standard DirectX .FX file.
base_mat - The base material type of this shader.

void EffectSetTechnique(int effect, string technique)
Description
Set the current technique for the effect. An effect can be composed by many techniques (for example: one with shaders 2.0, another with shaders 1.0, etc), and you can choose which one to use inorder to render the effect.

void EffectSetTexture(int effect, string var_name, int texture)
Description
Set an effect texture. A texture is a variable of type 'texture' defined in the effect. You can pass a texture created or loaded within ColdSteel to the effect using this function.

void EffectSetBool(int effect, string var_name, int value)
Description
Set the value of a boolean variable in the effect.

void EffectSetInt(int effect, string var_name, int value)
Description
Set the value of an integer variable in the effect.

void EffectSetFloat(int effect, string var_name, float value)
Description
Set the value of a floating point variable in the effect.

void EffectSetVector(int effect, string var_name, int vector)
Description
Set the value of a vector variable in the effect.

void EffectSetMatrix(int effect, string var_name, int matrix)
Description
Set the value of a matrix in the effect.
