Display

The display is the screen or window in which everything is drawn. You can create a fullscreen display, a windowed one, or even start a display in an already existing window.

Constants

DFE_RENDERTOTEXTURE - Render to texture support (render to texture will be included in a future update).
DFE_BILINEARFILTER - Display feature. Bilinear filtering of textures support.
DFE_HARDWARETNL - Display feature. Hardware Transform and Lighting support.
DFE_MIPMAP - Display feature. Texture mip-mapping support.
DFE_STENCILBUFFER - Display feature. Does this display have the stencil buffer enabled?
DFE_VERTEXSHADER11 - Display feature. Vertex shader 1.1 support.
DFE_VERTEXSHADER20 - Display feature. Vertex shader 2.0 support.
DFE_VERTEXSHADER30 - Display feature. Vertex shader 3.0 support.
DFE_PIXELSHADER11 - Display feature. Pixel shader 1.1 support.
DFE_PIXELSHADER12 - Display feature. Pixel shader 1.2 support.
DFE_PIXELSHADER13 - Display feature. Pixel shader 1.3 support.
DFE_PIXELSHADER14 - Display feature. Pixel shader 1.4 support.
DFE_PIXELSHADER20 - Display feature. Pixel shader 2.0 support.
DFE_PIXELSHADER30 - Display feature. Pixel shader 3.0 support.
DFE_ARB_VERTEXPROGRAM1 - Display feature. ARB vertex program 1.0 support.
DFE_ARB_FRAGMENTPROGRAM1 - Display feature. ARB fragment program 1.0 support.
DF_FULLSCREEN - Display flag. The display will be started in fullscreen mode instead of in a window
DF_VSYNC - Display flag. Screen refresh will be vertically synced. This is used to prevent from 'tearing' images, but limits the framerate to the vertical sync rate of the monitor.
DF_ANTIALIAS - Display flag. Enables fullscreen antialiasing.
DF_OPENGL - Display flag. The display will use OpenGL instead of Direct3D9.


Functions

void DisplayOpen(int width, int height, int depth, int flags, int win)
Description
Opens the ColdSteel display with the specified parameters.

Parameters
width, height - Dimensions of the display screen or window.
depth - Colour depth (in bits, 16, 24 or 32).
flags - Display flags. Can be a combination of DF_FULLSCREEN, DF_VSYNC, DF_ANTIALIAS, and DF_OPENGL.
win - Handle to an existing window (this is, a HWND in the Win32 API), or 0 to create a new one. This is useful when you are using the engine as a .dll (SDK version only), and you want to create and handle the application window, instead of letting ColdSteel do it.

void DisplayClose()
Description
Closes the ColdSteel display. Nothing may be rendered after doing this.

void DisplayCaption(string caption)
Description
Sets the title of the ColdSteel window.

Parameters
caption - Title to be shown in the window title bar.

int DisplayClosed()
Description
Tells if the display has been closed or not.

Returns
A value of '1' when the display does not exist, or '0' if it does.

int DisplayWidth()
Description
Returns the width of the ColdSteel display.

int DisplayHeight()
Description
Returns the height of the ColdSteel display.

int DisplayFps()
Description
Returns the frames per second at which ColdSteel is currently running.

int DisplayFeature(int feature)
Description
Tells if the specified feature is supported by the display.

Returns
Returns '1' if the feature is supported, '0' otherwise.

Parameters
feature - The feature you want to check. It can be one of the display feature constants explained above in this page.

void DisplayResize(int width, int height)
Description
If you are rendering on a custom window, and it has been resized, you can call this function to resize the rendering area to match the new dimensions.

int DisplayActive()
Description
Tells if the engine window is the currently active one.

void DisplayScreenshot(string file)
Description
Takes a screenshot and saves it as a bitmap.
