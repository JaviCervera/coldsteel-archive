Light

Lights are used to illuminate the scene. Current hardware only allows 8 lights in the scene at the same time.

Constants

LT_POINT - Point light. It has a position in space and emits light in all directions.
LT_DIRECTIONAL - Directional light. Similar to the sunlight. Comes from a direction in an infinite space.


Functions

int LightNode(int parent)
Description
Creates a new point light scene node.

Returns
The handle of the light node created.

Parameters
parent - Handle of another scene node, which will be the parent of the light. When the parent moves or rotates, the light will move and rotate with it to always maintain the same relative position with it. If you don't want the light to have a parent, set this to '0'.

void LightType(int light, int type)
Description
Set the type of the light, which can be point (default) or directional.
