Functions

void MousePosition(int x, int y)
Description
Positions the mouse pointer in the specified x, y device coordinates.

void MouseHide(int hide)
Description
If hide is '1', hides the mouse pointer. If it is '0', shows it.

int MouseX()
Description
Returns the device x coordinate of the mouse pointer.

int MouseY()
Description
Returns the device y coordinate of the mouse pointer.

int MouseHit(int button)
Description
Returns '1' if the specified button has been pressed, or '0' otherwise.

Parameters
button - The button you want to check. Use MB_LEFT, MB_RIGHT, or MB_MIDDLE.

int MouseDown(int button)
Description
Returns '1' if the specified button is pressed, or '0' otherwise.

Parameters
button - The button you want to check. Use MB_LEFT, MB_RIGHT, or MB_MIDDLE.

int MouseGet()
Description
Returns the number of the last button pressed.

int MouseReleased()
Description
Returns the number of the last button released.

int KeyHit(int key)
Description
Returns '1' if the specified key has been pressed, or '0' otherwise.

Parameters
key - The key you want to check. Each key has an associated constant value, named KEY_NAME. For example, KEY_UP, KEY_RETURN, KEY_ESC, KEY_A, KEY_6 and KEY_SPACE are valid values.

int KeyDown(int key)
Description
Returns '1' if the specified key is pressed, or '0' otherwise.

Parameters
key - The key you want to check. Each key has an associated constant value, named KEY_NAME. For example, KEY_UP, KEY_RETURN, KEY_ESC, KEY_A, KEY_6 and KEY_SPACE are valid values.

int KeyGet()
Description
Returns the last key pressed.

int KeyReleased()
Description
Returns the last key released.
