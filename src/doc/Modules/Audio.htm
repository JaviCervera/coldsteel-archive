Audio

This module controls the 2D and 3D audio library contained in ColdSteel.

Functions

void ListenerUpdate(int node)
Description
Updates the listener using a node position and rotation.

void SoundSetMasterVolume(float volume)
Description
Set the master sound volume, in a range between 0.0 and 1.0.

float SoundGetMasterVolume(float volume)
Description
Get the master sound volume, in a range between 0.0 and 1.0.

int SoundLoad(string file)
Description
Loads a sound and returns its handle. A sound can be in WAV or OGG format.

void SoundFree(int sound)
Description
Frees a previously loaded sound.

int SoundPlay2D(int sound, int loop)
Description
Plays a sound and returns a handle to the channel used to play it.

int SoundPlay3D(int sound, float x, float y, float z, int loop)
Description
Plays a sound in 3D space, and returns a handle to the channel used to play it.

int SoundIsPlaying(int sound)
Description
Returns '1' of the specified sound is playing, or '0' otherwise.

int ChannelPaused(int channel)
Description
Returns '1' if the specified channel has been paused, or '0' otherwise.

int ChannelFinished(int channel)
Description
Returns '1' if the specified channel has finished playing the sound, or '0' otherwise.

int ChannelLooped(int channel)
Description
Returns '1' if the specified channel is in loop mode, or '0' otherwise.

float ChannelGetMinDistance(int channel)
Description
Get the distance after which the sound played by this channel won't get any louder.

float ChannelGetMaxDistance(int channel)
Description
Get the distance after which the sound played by this channel won't get any quieter.

float ChannelGetPan(int channel)
Description
Returns the pan of the sound. Takes a value between '-1' and '1', '0' is center.

int ChannelGetPlayPosition(int channel)
Description
Returns the current play position of the sound in milliseconds.

float ChannelX(int channel)
Description
Returns the X position of the sound emitted by this channel in 3D space, if it is a 3D sound.

float ChannelY(int channel)
Description
Returns the Y position of the sound emitted by this channel in 3D space, if it is a 3D sound.

float ChannelZ(int channel)
Description
Returns the Z position of the sound emitted by this channel in 3D space, if it is a 3D sound.

float ChannelGetVolume(int channel)
Description
Returns the volume of the channel, between '0' and '1'.

void ChannelPause(int channel)
Description
Pauses a channel.

void ChannelResume(int channel)
Description
Resumes a channel which were previously paused.

void ChannelSetDistance(int channel, float min, float max)
Description
Set the min and max distance for a 3D sound channel. Further max distance, the sound won't be audible. Nearer min distance, it's heard at full volume.

void ChannelSetPan(int channel, float pan)
Description
Set the pan of the sound. Takes a value between '-1' and '1', '0' is center.

void ChannelSetPosition(int channel, float x, float y, float z)
Description
Set the position in 3D space of the sound emitted by this channel, if it is a 3D sound.

void ChannelSetVolume(int channel, float volume)
Description
Set the volume of the channel, between '0' and '1'.

void ChannelStop(int channel)
Description
Stop a channel and free its resources from memory.
